Добро пожаловать на наш проект!

Наш проект направлен на создание удобной веб-платформы, которая поможет людям с ограниченными возможностями здоровья получать необходимую помощь от волонтеров в городской среде.

Основная цель - обеспечить комфортное передвижение по городу для людей с ОВЗ, создав систему оперативного взаимодействия между нуждающимися в помощи и волонтерами.

О проекте

Цели проекта

  • Создание удобного инструмента для коммуникации между людьми с ОВЗ и волонтерами
  • Разработка интерактивной карты с возможностью отметки мест, где требуется помощь
  • Обеспечение безопасности и конфиденциальности пользователей
  • Создание системы мотивации для волонтеров

Функционал платформы

Для людей с ОВЗ:

  • Создание запроса на помощь
  • Просмотр доступных волонтеров
  • Оценка полученной помощи
  • Личный кабинет с историей запросов

Для волонтеров:

  • Просмотр актуальных запросов
  • Система уведомлений
  • Рейтинг и статистика
  • Система поощрений
